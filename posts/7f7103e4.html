Brief Intro to TACC

Study of efficient computation

1960s Hartmanis, Stearns - Time Hierarchy Theorem.

Turing machine, register based machine can simulate each other.

Turing Machines

Turing machines - simple, and has some nice properties:

  • Clean definition of complexity (time and space)
  • Straightforward to non-determinism and randomness.

Consider “offline multi-tape sequential Turing machine”

Turing machine is a finite-state machine that can access and modify chars written on “tapes” according to some rules.

Formal Definition

Turing machine MMM with one input tape and kkk work tapes. It includes:

  • Finite set QQQ: set of states

    • Element s∈Qs\in Qs∈Q: start state
    • Element yes∈Qyes\in Qyes∈Q: accept state
    • Element no∈Qno\in Qno∈Q: reject state

  • Finite set Γ\GammaΓ: the tape alphabet

    • non-empty finite set Σ⊆Γ\Sigma\subseteq\GammaΣ⊆Γ: input alphabet

  • Element Δ∈Γ╲Σ\Delta\in\Gamma\diagdown\SigmaΔ∈Γ╲Σ: blank symbol

  • Function

    δ:(Q╲{yes,no})×(Σ∪{Δ})×Γk→Q×(Γ╲{Δ})k×{L,S,R}k+1\delta:(Q\diagdown\{yes,no\})\times(\Sigma\cup\{\Delta\})\times\Gamma^k\rarr Q\times(\Gamma\diagdown\{\Delta\})^k\times\{L,S,R\}^{k+1}δ:(Q╲{yes,no})×(Σ∪{Δ})×Γk→Q×(Γ╲{Δ})k×{L,S,R}k+1

    is the transition function.

状态转移方程的意思：

转移前：(Q╲{yes,no})(Q\diagdown\{yes,no\})(Q╲{yes,no}) 代表转移前状态，不能是接受或拒绝态（因为接受或拒绝就停机了）；(Σ∪{Δ})(\Sigma\cup\{\Delta\})(Σ∪{Δ}) 代表输入字符；Γk\Gamma^kΓk 代表之前当前针头分别指 kkk 个工作纸带对应位置的字符。

转移后：QQQ 代表转移后的状态，可以是接受或拒绝态；(Γ╲{Δ})k(\Gamma\diagdown\{\Delta\})^k(Γ╲{Δ})k 表示针头分别在 kkk 个工作纸带上对应位置写入的字符；{L,S,R}k+1\{L,S,R\}^{k+1}{L,S,R}k+1 表示接下了输入纸带和 kkk 个工作纸带应该左移（LLL）、右移（RRR），还是不动（SSS）。

Transition Function has following properties:

  1. MMM will stop operation when entering yesyesyes or nonono state.
  2. MMM marks all cells visited on its work tapes

The operation of MMM:

  • k+1k+1k+1 two-way infinite tapes. One for input, the rest for work.
  • The cells of each work tape are numbered by the set Z\mathbb ZZ.
  • Input tape on which the cell 1,⋯ ,∣x∣1,\cdots,|x|1,⋯,∣x∣ are filled with char xix_ixi​. The other cells are filled with Δ\DeltaΔ.
  • Each tape has a tape head, which initially stays at position 000.
  • The machine MMM maintain an element of AAA as its internal state.

The computation of MMM:

  • Move the tapes according to transition function
  • If input head is called to “move left” at position 000, or “move right” at position ∣x∣+1|x|+1∣x∣+1, let MMM “crash”. (So there are ∣x∣+2|x|+2∣x∣+2 legal cell for input tape head)
  • When MMM enters yesyesyes or nonono, it stops. We called MMM halts. (M(x)=yesM(x)=yesM(x)=yes or M(x)=noM(x)=noM(x)=no)

Turing machine can be seen as language acceptor useful for decision problems whether the language is formal language, i.e., the language computed by MMM is L(M)={x∈Σ∣M(x)="yes"}L(M)=\{x\in\Sigma|M(x)="yes"\}L(M)={x∈Σ∣M(x)="yes"}.

An Extension on Turing Machine

Add an additional output tape which is write-only.

Λ\LambdaΛ: non-empty finite set, output alphabet. (Usually Λ=Σ\Lambda=\SigmaΛ=Σ)

In each step of computation MMM can choose to write a single symbol to the output tape. Then transition function:

δ:(Q╲{yes,no})×(Σ∪{Δ})×Γk→Q×(Γ╲{Δ})k×(Λ∪{ϵ})×{L,S,R}k+1\delta:(Q\diagdown\{yes,no\})\times(\Sigma\cup\{\Delta\})\times\Gamma^k\\ \rarr Q\times(\Gamma\diagdown\{\Delta\})^k\times(\Lambda\cup\{\epsilon\})\times\{L,S,R\}^{k+1}δ:(Q╲{yes,no})×(Σ∪{Δ})×Γk→Q×(Γ╲{Δ})k×(Λ∪{ϵ})×{L,S,R}k+1

ϵ\epsilonϵ here means write nothing.

Such a Turing machine is also called a transducer.

Time and Space Complexity

For a given input x∈Σ∗x\in\Sigma^*x∈Σ∗:

  • timeM(x)time_M(x)timeM​(x) is the number of computational steps that MMM performs on input xxx until halting.
  • spaceM(x)space_M(x)spaceM​(x) is the total number of cells on the work tapes that are at some point scanned by a tape head during computation on input xxx.

Both time and space can be infinite. Space can be infinite when MMM does not halt.

📖Definition of time/space-bounded: let T:N→NT:\mathbb N\rarr\mathbb NT:N→N and S:N→NS:\mathbb N\rarr\mathbb NS:N→N be functions with T(n)≥nT(n)\ge nT(n)≥n. We say MMM is T(n)T(n)T(n) time-bounded, if for every x∈Σ∗x\in\Sigma^*x∈Σ∗ we have timeM(x)≤T(∣x∣)time_M(x)\le T(|x|)timeM​(x)≤T(∣x∣). Similarly, MMM is S(n)S(n)S(n) space-bounded, if for every x∈Σ∗x\in\Sigma^*x∈Σ∗ we have spaceM(x)≤S(∣x∣)space_M(x)\le S(|x|)spaceM​(x)≤S(∣x∣).

Then define the first complexity classes based on time/space bounds.

📖Definition of DTIME,DSPACEDTIME,DSPACEDTIME,DSPACE: DTIME(T(n))DTIME(T(n))DTIME(T(n)) is the class of languages computed by a O(T(n))O(T(n))O(T(n)) time-bounded Turing machine and DSPACE(S(n))DSPACE(S(n))DSPACE(S(n)) is the class of languages computed by a O(S(n))O(S(n))O(S(n)) space Turing machine.

DDD 代表 deterministic （确定）。确定性图灵机每次状态转移只有一种。

NNN 代表 non-deterministic （非确定）。非确定性图灵机每次状态转移可能有多种，机器随机选择其中的一种进行转移。

Why OOO notation here?

  1. Compressing several tapes symbols to one class by an arbitrary linear factor.

  2. WLOG, one work tape can simulate kkk work tapes by special marking char.

    也就是说可以用一个标记字符把一个工作纸带分成 kkk 份。

📖Definition of time/space-constructible: TTT is time-constructible if there is a O(T(n))O(T(n))O(T(n)) time-bounded Turing machine computing the function 1n↦1T(n)1^n\mapsto 1^{T(n)}1n↦1T(n). SSS is space-constructible if there is a O(S(n))O(S(n))O(S(n)) time-bounded Turing machine computing the function 1n↦1S(n)1^n\mapsto 1^{S(n)}1n↦1S(n).

Universal Turing Machines

Universal Turing machine has a fixed amount of work tapes and a fixed tape alphabet.

Set of states QQQ is identified with positive integers from 111 to ∣Q∣|Q|∣Q∣.

Σ\SigmaΣ and Γ\GammaΓ is also identified with positive integers.

δ\deltaδ can be encoded by its function table.

Given: encoding of a Turing machine MMM with kkk work tapes with an encoding of an input xxx to MMM. (encoding of xxx is concatenation of each char in xxx).

🎯Theorem 1. For any k≥1k\ge1k≥1, there exist a Turing machine UkU_kUk​ with k+1k+1k+1 work tapes s.t. when given as input an encoding of a Turing machine MMM with kkk work tapes together with an encoding of an input xxx, simulates the computation of MMM on input xxx using time O(timeM(x))O(time_M(x))O(timeM​(x)) and space O(spaceM(x))O(space_M(x))O(spaceM​(x)).

“Simulates” here means Uk+1U_{k+1}Uk+1​ provides same output, terminal state as MMM.

Sublogarithmic Space

Usually assume that S(n)≥log⁡nS(n)\ge\log nS(n)≥logn.

What if S(n)=o(log⁡n)S(n)=o(\log n)S(n)=o(logn)? In this case Turing machine can’t even be able to describe the position of the input tape head.

O(1)O(1)O(1) space-bounded Turing machine is equivalent to a two-way finite automaton.

To accept non-regular language, the minimal space should be O(log⁡log⁡n)O(\log\log n)O(loglogn).

Storage configuration: a state of MMM, position of tape heads, non-black portions of the work tapes. # storage configuration: at most ∣Q∣sk∣Γ∣s+k−1=2O(s)|Q|s^k|\Gamma|^{s+k-1}=2^{O(s)}∣Q∣sk∣Γ∣s+k−1=2O(s).

🎯Theorem 2. Let MMM be a S(n)S(n)S(n) space-bounded Turing machine for a function S(n)=o(log⁡log⁡n)S(n)=o(\log\log n)S(n)=o(loglogn). Then MMM is in fact O(1)O(1)O(1) space-bounded.

Proof via contradiction.

Storage configuration is periodic.

MMM will produce at most 22O(s)2^{2^{O(s)}}22O(s) crossing sequences.

Turing Machine Configurations

Configuration 被翻译为“格局”……无力吐槽

A configuration of a Turing machine MMM with kkk work tapes:

  1. State of MMM
  2. Positions of k+1k+1k+1 tapes head
  3. Non-blank portions of the work tapes

Observation: If a Turing machine repeats a configuration, it will enter an infinite loop, infinitely repeating the computation taking place between two repeating configurations.

The Space Hierarchy Theorem

Idea: More resources means more can be computed.

🎯Theorem 3: Let S2(n)≥log⁡nS_2(n)\ge\log nS2​(n)≥logn be space constructible. If S1(n)=o(S2(n))S_1(n)=o(S_2(n))S1​(n)=o(S2​(n)), then

DSPACE(S1(n))⊊DSPACE(S2(n)).DSPACE(S_1(n))\subsetneq DSPACE(S_2(n)).DSPACE(S1​(n))⊊DSPACE(S2​(n)).

中文简单解释一下：图灵机的可用渐进存储空间越多，那么解决的问题也就越多。

It suffices to construct a O(S2(n))O(S_2(n))O(S2​(n)) space bounded Turing machine MMM s.t. L(M)∉DSPACE(S1(n))L(M)\not\in DSPACE(S_1(n))L(M)∈DSPACE(S1​(n)).

Input xxx of length n=∣x∣n=|x|n=∣x∣:

  1. Mark S2(n)S_2(n)S2​(n) cells of work tape using the space constructability of S2S_2S2​. During the remaining computation:
    • Ensuring that computation does not use more than S2(n)S_2(n)S2​(n) additional cells of work tape. This is done by immediately halting (reject).
    • Ensuring that computation does not run for more than 22S2(n)2^{2S_2(n)}22S2​(n) steps, by keeping time with a binary counter on 2S2(n)2S_2(n)2S2​(n) bits. (accept)
  2. Check if the input is on the form x=0i1wx=0^i1wx=0i1w, here www is an encoding of a Turing machine with a single work tape. Otherwise halt by rejecting.
  3. Simulate M′M'M′ on input xxx using the universal Turing machine U1U_1U1​. If this causes M′M'M′ to halt, halt and reject if M′M'M′ accepts. Otherwise accept.

Then analysis MMM and M′M'M′…

Tape Reduction and Oblivious Turing Machines

A T(n)T(n)T(n) time-bounded computation of a Turing machine with any number of work tapes can be simulated by a O(T(n)2)O(T(n)^2)O(T(n)2) time-bounded Turing machine with a single work tape. (Simple divide work tape into kkk tracks.)

This is used to prove DTIME(T1(n))⊊DTIME(T2(n))DTIME(T_1(n))\subsetneq DTIME(T_2(n))DTIME(T1​(n))⊊DTIME(T2​(n)) when T2(n)>nT_2(n)>nT2​(n)>n is time-constructible and (T1(n))2=o(T2(n))(T_1(n))^2=o(T_2(n))(T1​(n))2=o(T2​(n)).

📖Definition of oblivious: Let MMM be a Turing Machine. MMM is oblivious if for any n≥1n\ge1n≥1 and any input xxx of length n=∣x∣n=|x|n=∣x∣, during computation with input xxx and until MMM halts, the location of all tape heads of MMM is a function of nnn and the number ttt of steps of the computation so far.

It is easy to modify the O(T(n)2)O(T(n)^2)O(T(n)2) to be oblivious by first copying the input to an additional track, and thereafter keeping the input head stationary.

🎯Theorem 4: Let T(n)≥nT(n)\ge nT(n)≥n and suppose MMM is a T(n)T(n)T(n) time-bounded Turing machine with kkk work tapes. Then there exist a O(T(n)log⁡T(n))O(T(n)\log T(n))O(T(n)logT(n)) time-bounded oblivious Turing machine M′M'M′ with 222 work tapes s.t. L(M)=L(M′)L(M)=L(M')L(M)=L(M′).

Hard proof.

The Time Hierarchy Theorem

🎯Theorem 5: Let T2(n)≥nT_2(n)\ge nT2​(n)≥n be time constructible. If T1(n)≥nT_1(n)\ge nT1​(n)≥n satisfies T1(n)log⁡T1(n)=o(T2(n))T_1(n)\log T_1(n)=o(T_2(n))T1​(n)logT1​(n)=o(T2​(n)), then

DTIME(T1(n))⊊DTIME(T2(n)).DTIME(T_1(n))\subsetneq DTIME(T_2(n)).DTIME(T1​(n))⊊DTIME(T2​(n)).

Proof by tape reduction.

There are some standard complexity classes given by natural classes of time and space bounds.
