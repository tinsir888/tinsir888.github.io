Modern Algebra

Magma

magma 代数系统

Binary operation assume S is a set mapping f: S×S→SS\times S\rightarrow SS×S→S is a dyadic operation on S.

dyadic operation 二元运算:

  1. The result of the dyadic operation is still belong to S.

  2. Every element in S can participate in the dyadic operation and get a unique result.

    e.g. In the natural number set N addition and multiplication are binary operators on N, but subtraction and division are not.

Quasigroup

准群

assume G is a nonempty set and ∘\circ∘ is the dyadic operation on the G, then ∘\circ∘ is called multiplication. If the following condition is true, we call G is a quasigroup about binary operation multiplication ∘\circ∘.

a,b∈Ga,b\in Ga,b∈G, equation a∘x=ba\circ x=ba∘x=b and y∘a=by\circ a=by∘a=b have solutions in G.

Semigroup

半群

assume G is a nonempty set and ∘\circ∘ is the dyadic operation on the G, ∘\circ∘ is called multiplication. If the following condition is true, we call G is a semigroup about dyadic operation multiplication ∘\circ∘.

a,b,c∈Ga,b,c\in Ga,b,c∈G, (a∘b)∘c=a∘(b∘c)(a\circ b)\circ c=a\circ(b\circ c)(a∘b)∘c=a∘(b∘c). (associative law)

Monoid

幺半群

Suppose that S is a set and ⋅\cdot⋅ is some dyadic operation in S, then S with ⋅\cdot⋅ is a monoid if it satisfies the following two axioms

  1. Associativity for all a, b and c in S, the equation (a⋅b)⋅c=a⋅(b⋅c)(a\cdot b)\cdot c=a\cdot(b\cdot c)(a⋅b)⋅c=a⋅(b⋅c) holds.
  2. Identity element. there exists an element e in S such that for every element a in S, equation e⋅a=a⋅ee\cdot a=a\cdot ee⋅a=a⋅e hold.

Group

群

A group is a set, G, together with an operation ⋅\cdot⋅ that combines any two elements a and b to form another element, denoted a⋅ba\cdot ba⋅b or ababab. To qualify as a group, the set and operation, (G,⋅)(G,\cdot)(G,⋅), must satisfy 4 requirements known as the group axioms 群公理

  1. Closure for all a, b in G, the result of the operation a⋅ba\cdot ba⋅b, is also in G.
  2. Associativity. for all a, b and c in G, (a⋅b)⋅c=a⋅(b⋅c)(a\cdot b)\cdot c=a\cdot(b\cdot c)(a⋅b)⋅c=a⋅(b⋅c).
  3. Identity element. there exists an element e in S such that for every element a in S, equation e⋅a=a⋅ee\cdot a=a\cdot ee⋅a=a⋅e hold.
  4. Inverse element. for each a in G, there exists an element b in G, commonly denoted a−1a^{-1}a−1 or −a-a−a, s.t. a⋅b=b⋅a=ea\cdot b=b\cdot a=ea⋅b=b⋅a=e, where e is the identity element.

Ring

A ring is a set R equipped with 2 binary operations +++ and ⋅\cdot⋅, satisfying the following 3 sets of axioms, called the ring axioms.

  1. R is an abelian group（阿贝尔群） under addition. i.e.
    • (a+b)+c=a+(b+c)(a+b)+c=a+(b+c)(a+b)+c=a+(b+c) associative
    • a+b=b+aa+b=b+aa+b=b+a commutative 交换律
    • there is an element 0 in R s.t. a+0=aa+0=aa+0=a for all a in R (0 is additive identity)
    • for each a in R there exists -a in R s.t. a+(−a)=0a+(-a)=0a+(−a)=0 (-a is the additive inverse of a) 加法逆元
  2. R is a monoid under multiplication.
    • (a⋅b)⋅c=a⋅(b⋅c)(a\cdot b)\cdot c=a\cdot(b\cdot c)(a⋅b)⋅c=a⋅(b⋅c) associative
    • a⋅1=1⋅a=aa\cdot1=1\cdot a=aa⋅1=1⋅a=a multiplicative identity
  3. Multiplication is distributive with respect to addition.
    • a⋅(b+c)=(a⋅b)+(a⋅c)a\cdot(b+c)=(a\cdot b)+(a\cdot c)a⋅(b+c)=(a⋅b)+(a⋅c) left distributivity
    • (b+c)⋅a=(b⋅a)+(c⋅a)(b+c)\cdot a=(b\cdot a)+(c\cdot a)(b+c)⋅a=(b⋅a)+(c⋅a) right distributivity
