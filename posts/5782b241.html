Boolean Branching Program

Deterministic

📖Definition 32. A Deterministic Boolean branching program BBB with nnn inputs consists of the following:

  • A directed acyclic graph D=(V,A)D=(V,A)D=(V,A), where the out-degree of all nodes is in {0,1,2}\{0,1,2\}{0,1,2}.
  • A partial labeling ℓV:V→{0,1,x1,⋯ ,xn}\ell_V:V\to\{0,1,x_1,\cdots,x_n\}ℓV​:V→{0,1,x1​,⋯,xn​} of the nodes of DDD.
  • A partial labeling ℓA:A→{0,1}\ell_A:A\to\{0,1\}ℓA​:A→{0,1} of the edges of DDD.
  • An initial node s∈Vs\in Vs∈V.

When node vvv is not labeled by ℓV\ell_VℓV​, we say that vvv is dummy.

If ℓV(v)∈{0,1}\ell_V(v)\in\{0,1\}ℓV​(v)∈{0,1}, we say that vvv is an output node.

The graph DDD and the labeling function ℓV,ℓA\ell_V,\ell_AℓV​,ℓA​ should satisfy:

  1. The node sss is not a dummy node.
  2. If ℓV(v)∈{x1,⋯ ,xn}\ell_V(v)\in\{x_1,\cdots,x_n\}ℓV​(v)∈{x1​,⋯,xn​}, the out-degree of vvv is 222, and one outgoing edges of vvv is labeled 000 and 111 by ℓA\ell_AℓA​ respectively.
  3. If ℓV(v)∈{0,1}\ell_V(v)\in\{0,1\}ℓV​(v)∈{0,1}, i.e., vvv is an output node, the out-degree of vvv is 000.
  4. If vvv is a dummy node, the in-degree of vvv is not 000, the out-degree of vvv is 111.

A branching program with nnn inputs compute a Boolean function f:{0,1}n→{0,1}f:\{0,1\}^n\to\{0,1\}f:{0,1}n→{0,1} in a natural way.

Given an input x∈{0,1}nx\in\{0,1\}^nx∈{0,1}n, start in the initial node sss, follow a path to an output node, thereby giving the value of f(x)f(x)f(x).

When passing through dummy node, directly go through the unique path.

When passing through labeled node, go to 111-labeled edge if xi=1x_i=1xi​=1, 000-labeled edge if xi=0x_i=0xi​=0.

Non-deterministic

Simply modify the deterministic branching program:

  • label of vertices is ℓV:V→{0,1,∗,x1,⋯ ,xn}\ell_V:V\to\{0,1,*,x_1,\cdots,x_n\}ℓV​:V→{0,1,∗,x1​,⋯,xn​}. If ℓV(v)=∗\ell_V(v)=*ℓV​(v)=∗, vvv is called a choice node.
  • We require the out-degree of vvv is 222 and outgoing edges are not labeled by ℓA\ell_AℓA​.

For a given input x∈{0,1}nx\in\{0,1\}^nx∈{0,1}n, traverse the branching program from sss, but at a choice node choose to follow an arbitrary arc.

We say that the non-deterministic branching program accepts xxx if there exist such choices leading the path to an output node labeled 111.

📖Definition 33. The size of a branching program is the number of nodes in the branching program. The length of a branching program is the length of the longest path from sss to a node to out degree 000.

Then define complexity classes of languages computed by size-bounded branching programs.
