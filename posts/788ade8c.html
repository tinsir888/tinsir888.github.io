数据挖掘 6 Spectral Theory and Clustering

This module introduces graph mining.

Data points from networks, graphs, of connections.

Purpose: extract patterns from graphs.

3 different areas of graph mining:

  1. Community detection: finding groups of nodes with similar characteristics.
  2. Link analysis: understanding which node is more important than others.
  3. Embedding: capturing similarities among nodes.

Graph G=(V,E)G=(V,E)G=(V,E): a set of vertices VVV connected through edges E⊆V×VE\subseteq V\times VE⊆V×V.

undirected & directed graph

Graph representations. A graph can be represented by adjacency lists or an adjacency matrix.

Degree of a node.

Graph construction:

  • KNN graph: connect every point x\mathbb xx with its closest kkk points.
  • ϵ\epsilonϵ-neighborhood: connect all points x\mathbb xx with the points within ϵ\epsilonϵ distance.
  • Fully-connected graph: choose a distance distdistdist, then connect any point to any other point. The weight of each edge (i,j)(i,j)(i,j) corresponds to 1−dist1-dist1−dist between iii and jjj.

The Adjacency Matrix

Consider undirected graphs. Their adjacency matrix is symmetric.

By multiplying a vector x∈Rn\mathbf x\in\mathbb R^nx∈Rn with a adjacency matrix.

x\mathbf xx can be thought as the vector form of a function f:V→Rf:V\rarr\mathbb Rf:V→R.

Ax=y⇒(a1,1⋯a1,n⋮⋱⋮an,1⋯an,n)(x1⋮xn)=(y1⋮yn)\mathbf{Ax}=\mathbf{y}\\ \Rarr \left( \begin{array}{cc}a_{1,1} & \cdots & a_{1,n}\\ \vdots & \ddots & \vdots\\ a_{n,1} & \cdots& a_{n,n} \end{array} \right) \left({\begin{array}{cc}x_1\\ \vdots\\x_n\end{array}}\right) = \left({\begin{array}{cc}y_1\\ \vdots\\y_n\end{array}}\right)Ax=y⇒⎝⎜⎜⎛​a1,1​⋮an,1​​⋯⋱⋯​a1,n​⋮an,n​​⎠⎟⎟⎞​⎝⎜⎜⎛​x1​⋮xn​​⎠⎟⎟⎞​=⎝⎜⎜⎛​y1​⋮yn​​⎠⎟⎟⎞​

The vector y\mathbf yy’s iii-th element is yi=∑(i,j)∈Exjy_i=\sum_{(i,j)\in E}x_jyi​=∑(i,j)∈E​xj​. Therefore, the entry yiy_iyi​ is the sum of labels xjx_jxj​ of neighbors of iii. This means that A\mathbf AA is a matrix that transforms the value of a node as a sum of the value of the neighbors. If we repeat this aggregation multiple times we obtain a propagation of the initial labels x\mathbf xx over the network x(t)=Ax(t−1)\mathbf x^{(t)}=\mathbf{Ax}^{(t-1)}x(t)=Ax(t−1)

There exists a fixed-point in this iteration:

∃λ,Ax=λx\exist\lambda,\mathbf{Ax}=\lambda\mathbf{x}∃λ,Ax=λx

We want to find an eigenvector x\mathbf xx and eigenvalue λ\lambdaλ for the matrix A\mathbf AA.

Spectral Graph Theory

Analyzing properties of the graph by inspecting the eigenvalues and eigenvectors of matrices that represent the graph structure.

Given a graph GGG and a matrix M\mathbf MM that describes the graph’s structure, the graph spectrum is the set of M\mathbf MM’s eigenvalues Λ={λ1,⋯ ,λn}\Lambda=\{\lambda_1,\cdots,\lambda_n\}Λ={λ1​,⋯,λn​} sorted in descending order λ1≥λ≥⋯≥λn\lambda_1\ge\lambda\ge\cdots\ge\lambda_nλ1​≥λ≥⋯≥λn​

λ∈C1\lambda\in\mathbb C^1λ∈C1 is an eigenvalue if there exists a vector x∈Cn,x≠0\mathbf x\in\mathbb C^n,\mathbf x\neq0x∈Cn,x=0 s.t. \mathbf{Ax}=\lambda\mathbf x\Lrarr(\mathbf A-\lambda\mathbf I)\mathbf x=0\Rarr\mathbf\det(A-\lambda\mathbf I)=0

For symmetric matrices. A\mathbf AA is symmetric, This means aj,i=ai,j∀i,ja_{j,i}=a_{i,j}\forall i,jaj,i​=ai,j​∀i,j. Thus,

  • All eigenvalues are positive λ≥0\lambda\ge0λ≥0

  • The matrix is positive semi-definite, x⊤Ax≥0∀x\mathbf x^\top\mathbf{Ax}\ge0\forall\mathbf xx⊤Ax≥0∀x.

  • ∃\exist∃ matrix N\mathbf NN, s.t. A=N⊤N\mathbf A=\mathbf N^\top\mathbf NA=N⊤N

  • All eigenvalues are real numbers, all eigenvectors are orthogonal (u⊤v=0\mathbf u^\top\mathbf v=0u⊤v=0).

In a ddd-regular graph, that 1\mathbf 11 is an eigenvector. That means all the eigenvectors u\mathbf uu should be 1⊤u=0⇒∑iui=0\mathbf 1^\top\mathbf u=0\Rarr\sum_iu_i=01⊤u=0⇒∑i​ui​=0. This is a very important fact that will be used later.

A graph is ddd-regular, if the degree for every vertex is ddd.

Spectral Clustering

A cut is a set of edges with only one vertex in a group

W(A,B)=∑i∈A,j∈Bai,jW(A,B)=\sum_{i\in A,j\in B}a_{i,j}W(A,B)=i∈A,j∈B∑​ai,j​

If we directly minimize the cut we might end up in unfortunate solution in which a partition contains only one node. To avoid this, we can minimize a normalized cut

Jrc(C)=W(A,B)∣A∣+W(A,B)∣B∣J_{rc}(C)=\frac{W(A,B)}{|A|}+\frac{W(A,B)}{|B|}Jrc​(C)=∣A∣W(A,B)​+∣B∣W(A,B)​

Where C={A,B}C=\{A,B\}C={A,B}. Computing optimal normalized cuts is NP-hard.

Another approach to write the cut is considering a variable xix_ixi​ for each vertex iii, s.t.

xi={+1  (i∈A)−1  (i∈B)x_i=\begin{cases} +1\ \ (i\in A)\\ -1\ \ (i\in B) \end{cases}xi​={+1  (i∈A)−1  (i∈B)​

Thus W(A,B)=∑(i,j)∈E∣xi−xj∣W(A,B)=\sum_{(i,j)\in E}|x_i-x_j|W(A,B)=∑(i,j)∈E​∣xi​−xj​∣.

So if we want to minimize the cut we can just minimize the sum of absolute differences among x\mathbf xx entries. It’s more convenient instead to minimize ∑i,j∈E(xi−xj)2\sum_{i,j\in E}(x_i-x_j)^2∑i,j∈E​(xi​−xj​)2. However, as we have seen this corresponds to x⊤Lx\mathbf x^\top L\mathbf xx⊤Lx.

Similarly, we can see that

x⊤Lxx⊤x\frac{\mathbf x^\top L\mathbf x}{\mathbf x^\top\mathbf x}x⊤xx⊤Lx​

corresponds to a normalized version of the cut and is called the Rayleigh quotient. （瑞利商）

Variantional characterization of eigenvalues. Consider a matrix M\mathbf MM real and symmetric. The eigenvalues are sorted in decreasing order. The variational characterization of the eigenvalues states that states that the smallest eigenvalue corresponds to

λn−1=min⁡x≠0,x⊤x1=0x⊤Mxx⊤x\lambda_{n-1}=\min_{\mathbf x\neq0,\mathbf x^\top\mathbf x_1=0}\frac{\mathbf x^\top\mathbf{Mx}}{\mathbf x^\top\mathbf x}λn−1​=x=0,x⊤x1​=0min​x⊤xx⊤Mx​

And so on with all the eigenvalues. Finally, λ1\lambda_1λ1​

λ1=max⁡x≠0x⊤Mxx⊤x\lambda_1=\max_{\mathbf x\neq0}\frac{\mathbf x^\top\mathbf{Mx}}{\mathbf x^\top\mathbf x}λ1​=x=0max​x⊤xx⊤Mx​

Minimum cut as eigenvalues of the Laplacian.

  • All eigenvectors a symmetric matrix are orthogonal and that the Laplacian matrix for an undirected graph is a symmetric matrix.
  • The eigenvalue that minimizes λn\lambda_nλn​ is the eigenvalue 000 that corresponds to the eigenvector 1\mathbf 11 of the graph’s Laplacian LLL. This corresponds to placing all nodes into a single partition.
  • The second smallest λn−1\lambda_{n-1}λn−1​ corresponds to an eigenvector x\mathbf xx, which is orthogonal to the eigenvector 1\mathbf 11, i.e. ∑ixi=0\sum_ix_i=0∑i​xi​=0.
  • The eigenvector is called the Fiedler’s vector and minimizes a relaxed version of the normalized cut.

Spectral Clustering Algorithm

Spectral clustering works in 3 phases:

  1. Preprocessing: Construct a matrix representation of the graph.
  2. Decomposition: compute the eigenvalues and eigenvectors of the matrix. Map each point to a lower-dimensional representation based on one or more eigenvectors.
  3. Grouping: Assign points to two or more clusters, based on the new representation.

The Spectral Partitioning Algorithm finds a partition of the graph that is close to the optimal partition. In the decomposition phase, Spectral clustering maps vertices to corresponding components of λn−1\lambda_{n-1}λn−1​ and its vector. In the grouping phase, we sort the components of the new 1-dimensional vector and split then nodes in positive and negative or using the median value.

kkk-way Spectral Clustering

We extend spectral clustering to the case of finding kkk-partitions.

One strategy is to recursively split the graphs into two partitions, until kkk partitions are returned.

Another strategy leverages multiple eigenvectors besides the one corresponding to the second smallest eigenvalue. By stacking on the eigenvectors each node is embedded into a multi-dimensional space. Finally, we run kkk-means in this new space to find kkk-partitions.

Advantages of using multiple eigenvectors:

  • Approximates the optimal cut
  • Emphasizes cohesive clusters by increasing the unevenness in the distribution in the data.
  • Provides a well-separated space. It transforms data to a new embedded space consisting of kkk orthogonal basis vectors.
  • Multiple eigenvectors prevent instability caused by information loss.

Pros and Cons

👍

  • Clusters can have arbitrary shapes and size and are not restricted to convex shapes.
  • Efficient in normal space graphs.
  • Robust to noise is theoretically grounded and well-connected to graph properties.

👎

  • Inefficient on dense graph - O(n3)O(n^3)O(n3) worst case.
  • Need to provide number of clusters kkk.
