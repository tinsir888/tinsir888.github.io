Non-uniform model of computation: Boolean circuit model.

In contrast to uniform model like Turing machine.

Boolean circuit: Fixed number of inputs. To study computation of arbitrary inputs, we need a family of Boolean circuits.

Some Definitions

📖Definition of Boolean circuit. A Boolean circuit CCC with nnn inputs and mmm outputs consists of:

  1. A directed acyclic graph D=(V,A)D=(V,A)D=(V,A) where the in-degree of every node is at most 222.
  2. A labeling ℓ\ellℓ of all nodes g∈Vg\in Vg∈V with \ell(g)\in\{0,1,x_1,\cdots,x_n,\overline{x_1},\cdots,\overline{x_n},\and,\or,\neg\}.
  3. A choice of mmm output gates o1,⋯ ,om∈Vo_1,\cdots, o_m\in Vo1​,⋯,om​∈V.

The nodes of DDD is the gates of CCC, the arcs of DDD is the wires of CCC.

Nodes with in-degree 000 are input gates.

The size of CCC is the number of gates.

Fanin of a gate is its in-degree, fanout is its out-degree.

Labeling function should satisfy: For all gates ggg:

  1. If the fanin is 000, then \ell(g)\in\{0,1,x_1,\cdots,x_n,\overline{x_1},\cdots,\overline{x_n},\and,\or,\neg\}.
  2. If the fanin is 111, then ℓ(g)=¬\ell(g)=\negℓ(g)=¬.
  3. If the fanin is 222, then \ell(g)\in\{\and,\or\}.

A Boolean circuit CCC with nnn inputs and mmm outputs computes a Boolean function f:{0,1}n→{0,1}mf:\{0,1\}^n\to\{0,1\}^mf:{0,1}n→{0,1}m.

Implicit negation such as xi‾\overline{x_i}xi​​ can be made explicit by introducing a new gate, thereby at most doubling the size of the circuit.

The Shannon Function

The smallest size L(n)L(n)L(n) in which any Boolean function on nnn variables can be computed by a Boolean circuit of size L(n)L(n)L(n).

The function L(n)L(n)L(n) is called the Shannon function.

By DNF/CNF formulas, we have trivial bound L(n)≤n2nL(n)\le n2^nL(n)≤n2n.

Shannon proved that L(n)=Θ(2n/n)L(n)=\Theta(2^n/n)L(n)=Θ(2n/n).

Turing Machines versus Boolean Circuits

Oblivious Turing machine can efficiently be converted to a family of Boolean circuits.

🤔Proposition 1. Let T(n)≥nT(n)\ge nT(n)≥n and suppose LLL is computed by a O(T(n))O(T(n))O(T(n)) time-bounded oblivious Turing machine. Then L∈SIZE(T(n))L\in SIZE(T(n))L∈SIZE(T(n)).

Any configuration of oblivious Turing machine on input xxx can be encoded by a binary string of length O(T(n))O(T(n))O(T(n)) s.t. every symbol of every work tape corresponds to a fixed number of positions of the encoding.

We can build the Boolean circuit for input length nnn in O(T(n))O(T(n))O(T(n)) steps. Proof by induction.

Proposition 1 and Theorem 4 result in some inclusion relation.
