MAX SAT and MAX CUT

randomized ½ approximation for each problem

MAX SAT

aka maximum satisfiability problem

  • Input consists of nnn Boolean variables x1,⋯ ,xnx_1,\cdots,x_nx1​,⋯,xn​, mmm clauses C1,⋯ ,CmC_1,\cdots,C_mC1​,⋯,Cm​.
    • Clauses CjC_jCj​ consist of some number of variables and \or and a non-negative weight wjw_jwj​.
  • The objective of the problem is to find an assignment of xix_ixi​ that maximizes the weight of the satisfied clauses.
  • A clause is said to be satisfied if one of the unnegated variables is set to true, or one of the negated variables is set to false.

Some terminology:

  • literal. A variable xix_ixi​ or a negated variable xˉi\bar x_ixˉi​ is a literal, so that each clause consists of some number of literals.
    • xix_ixi​ is called positive literal
    • xˉi\bar x_ixˉi​ is called negative literal
  • length. The number of literals in a clause. Length of CiC_iCi​ is lil_ili​.
  • Clause of length 1 is unit clause.

WLOG, assume that no literal is repeated in a clause, at most one of xix_ixi​ and xˉi\bar x_ixˉi​ appears in a clause.

Assume the clauses are distinct.

A very straightforward use of randomization: set each xix_ixi​ to true independently with probability 1/21/21/2.

🤔Theorem 1. Setting each xix_ixi​ to true with probability 1/21/21/2 independently gives a randomized 1/21/21/2-approximation algorithm for the MAX SAT problem.

Define indicator variable YjY_jYj​: 111 if clause jjj is satisfied and 000 otherwise.

Let WWW be a random variable that is equal to the total weight of the satisfied clauses, so that W=∑jwjYjW=\sum_jw_jY_jW=∑j​wj​Yj​.

Let OPTOPTOPT denote the optimum value of the MAX SAT instance. Then, by linearity of expectation, and the definition of the expectation of 0-1 random variable, we know

E[W]=∑j=1mwjE[Yj]=∑j=1mwjPr⁡[Cj is satisfied].\mathbb E[W]=\sum_{j=1}^mw_j\mathbb E[Y_j]=\sum_{j=1}^mw_j\Pr[C_j\text{ is satisfied}].E[W]=j=1∑m​wj​E[Yj​]=j=1∑m​wj​Pr[Cj​ is satisfied].

For each CjC_jCj​, the probability that it is not satisfied is the probability that each positive literal in CjC_jCj​ is set to false and each negative literal is set to true, each of which happens with probability 1/21/21/2. Hence,

Pr⁡[Cj is satisfied]=(1−(12)lj)≥12,\Pr[C_j\text{ is satisfied}]=\left(1-(\frac{1}{2})^{l_j}\right)\ge\frac{1}{2},Pr[Cj​ is satisfied]=(1−(21​)lj​)≥21​,

where the last inequality is a consequence of the fact that lj≥1l_j\ge1lj​≥1. Hence,

E[W]≥12∑j=1mwj≥12OPT,\mathbb E[W]\ge\frac{1}{2}\sum_{j=1}^mw_j\ge\frac{1}{2}OPT,E[W]≥21​j=1∑m​wj​≥21​OPT,

where the last inequality follows from the fact that the total weight is an easy upper bound on the optimal value, since each weight is assumed to be non-negative.

If lj≥kl_j\ge klj​≥k for each clause jjj, the algorithm is a (1−(12)k)\left(1-(\frac{1}{2})^k\right)(1−(21​)k)-approximation algorithm for such instances. Thus the longer the clauses, the better the approximation.

Consider lj=3l_j=3lj​=3 for all clauses, the problem turns into MAX E3SAT. By analysis above, this randomized algorithm gives guarantee of 7/87/87/8 approximation. Nothing better is possible for these instances unless P=NPP=NPP=NP.

MAX CUT

aka maximum cut problem

In this problem. the input is an undirected graph G=(V,E)G=(V,E)G=(V,E), along with a non-negative weight wij≥0w_{ij}\ge0wij​≥0 for each edge (i,j)∈E(i,j)\in E(i,j)∈E. The goal is to partition the vertex set into two parts UUU and W=V−UW=V-UW=V−U, so as to maximize the weight of edges whose two endpoints are in different parts, one in UUU and another in WWW.

An edge with endpoints in both UUU and WWW is in the cut. In the case wij=1w_{ij}=1wij​=1 for each edge ∈E\in E∈E, we have an unweighted MAX CUT problem.

We place each vertex v∈Vv\in Vv∈V into UUU independently with probability 1/21/21/2. As with the MAX SAT algorithm, this can be viewed as sampling a solution uniformly form the space of all possible solutions.

🤔Theorem 3. If we place each vertex v∈Vv\in Vv∈V into UUU independently with probability 1/21/21/2, then we obtain a randomized 1/21/21/2-approximation algorithm for the maximum cut problem.

Consider a random variable XijX_{ij}Xij​ that is 111 if edge (i,j)(i,j)(i,j) is in the cut, and 000 otherwise.

Let ZZZ be the random variable equal to the total weight of edges in the cut, so that Z=∑(i,j)∈EwijXijZ=\sum_{(i,j)\in E}w_{ij}X_{ij}Z=∑(i,j)∈E​wij​Xij​.

Let OPTOPTOPT denote the optimal value of the maximum cut instance. Then, as before, by linearity of expectation and the definition of expectation of a 0-1 random variable, we get that

E[Z]=∑(i,j)∈EwijE[Xij]=∑(i,j)∈EwijPr⁡[(i,j) in cut].\mathbb E[Z]=\sum_{(i,j)\in E}w_{ij}\mathbb E[X_{ij}]=\sum_{(i,j)\in E}w_{ij}\Pr[(i,j)\text{ in cut}].E[Z]=(i,j)∈E∑​wij​E[Xij​]=(i,j)∈E∑​wij​Pr[(i,j) in cut].

The probability that a specific edge (i,j)(i,j)(i,j) is in the cut: since the 2 endpoints are placed in the sets independently, they are in different sets with probability equal to 1/21/21/2. Hence,

E[Z]=12∑(i,j)∈Ewij≥12OPT,\mathbb E[Z]=\frac{1}{2}\sum_{(i,j)\in E}w_{ij}\ge\frac{1}{2}OPT,E[Z]=21​(i,j)∈E∑​wij​≥21​OPT,

where the inequality follows directly from the fact that the sum of the non-negative weights of all edges is obviously an upper bound on the weight of the edges in an optimal cut.

Derandomization

Randomized algorithm for MAX SAT can be derandomized by replacing the randomized decision of whether to set xix_ixi​ to true with a deterministic one that will preserve the expected value of the solution. The decisions will be made sequentially.

Assume that x1x_1x1​ is fixed, and other variables will be set true with probability 1/21/21/2. Intuitively we should choose the value of x1x_1x1​ which maximize the expected value of WWW. In this way, we should maintain the algorithm that the expected value is at least half the optimum, while having fewer random variables left.

If E[W∣x1←true]≥E[W∣x1←false]\mathbb E[W|x_1\gets true]\ge\mathbb E[W|x_1\gets false]E[W∣x1​←true]≥E[W∣x1​←false], then we set x1x_1x1​ to true, otherwise false. Then

E[W]=E[W∣x1←ture]Pr⁡[x1←true]+E[W∣x←false]Pr⁡[x1←flase]=12(E[W∣x1←true]+E[W∣x1←false])\mathbb E[W]=\mathbb E[W|x_1\gets ture]\Pr[x_1\gets true]+\mathbb E[W|x\gets false]\Pr[x_1\gets flase]\\ =\frac{1}{2}(\mathbb E[W|x_1\gets true]+\mathbb E[W|x_1\gets false])E[W]=E[W∣x1​←ture]Pr[x1​←true]+E[W∣x←false]Pr[x1​←flase]=21​(E[W∣x1​←true]+E[W∣x1​←false])

By induction, we can set the sequence of variables better than 1/21/21/2.

Randomized Rounding

Biasing probability with which we set xix_ixi​ true yields an improved approximation algorithm. However, we gave each variable the same bias. We can do better by giving each variable its own bias!

Idea of randomized rounding:

  • First set up an integer programming formulation of the problem at hand in which there are 0-1 integer variables. 0-1 variable yiy_iyi​ for each Boolean variable xix_ixi​ s.t. yi=1y_i=1yi​=1 corresponds to xix_ixi​ set true.
  • The integer program is relaxed to a linear program by replacing the constraints yi∈{0,1}y_i\in\{0,1\}yi​∈{0,1} with 0≤yi≤10\le y_i\le 10≤yi​≤1.
  • The fractional value yi∗y_i^*yi∗​ is interpreted as the probability that yiy_iyi​ should be set to 111. In this case we set each xix_ixi​ to true with probability yi∗y_i^*yi∗​ independently.

For MAX SAT problem, in addition to the variables yiy_iyi​, we introduce a variable zjz_jzj​ for each CjC_jCj​ that will be 111 if the clause is satisfied and 000 otherwise. For each clause CjC_jCj​ let PjP_jPj​ be the indices of the variables xix_ixi​ that are positive, NjN_jNj​ be negated in the clause.

We denote the clause CjC_jCj​ by

\bigvee_{i\in P_j}x_i\or\bigvee_{i\in N_j}\bar x_i.

Then the inequality

∑i∈Pjyi+∑i∈Nj(1−yi)≥zj\sum_{i\in P_j}y_i+\sum_{i\in N_j}(1-y_i)\ge z_ji∈Pj​∑​yi​+i∈Nj​∑​(1−yi​)≥zj​

must hold for CjC_jCj​ since if each variable that occurs positively in the clause is set to false and each variable that occurs negatively is set to true, then the clause is not satisfied, and zjz_jzj​ must be 000.

This yields integer programming formulation of MAX SAT problem:

  • maximize ∑j=1mwjzj\sum_{j=1}^mw_jz_j∑j=1m​wj​zj​
  • subject to ∑i∈Pjyi+∑i∈Nj(1−yi)≥zj\sum_{i\in P_j}y_i+\sum_{i\in N_j}(1-y_i)\ge z_j∑i∈Pj​​yi​+∑i∈Nj​​(1−yi​)≥zj​, \forall C_j=\bigvee_{i\in P_j}x_i\or\bigvee_{i\in N_j}\bar x_i,

Choosing the Better of 2 Solutions

Consider a given clause CjC_jCj​ of length ljl_jlj​.
