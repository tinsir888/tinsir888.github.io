Orthogonal Problems

aka 正交问题

input: set PPP of nnn points

goal: process PPP in a DS​

query: axis-aligned rectangle rrr

orthogonal range counting: number of points in rrr

orthogonal range reporting: list of points in rrr

3-sided queries: 3 boundaries

Rectangle stabbing

aka 穿刺矩形

input: set PPP of nnn rectangles

goal: process PPP in a DS

query: a point qqq

output: list of rectangles containing qqq

1D Orthogonal Range Searching

input: set PPP of nnn points in 1D

goal: process PPP in a DS

query: axis-aligned rectangle r→r\rightarrowr→ an interval

solution

balanced BST + linked list

O(n)O(n)O(n) space and O(log⁡n)O(\log n)O(logn) time to count, O(log⁡n+k)O(\log n+k)O(logn+k) time to report

query interval covered by log⁡n\log nlogn canonical sets 规范集.

可以证明任意一个区间包含的规范集个数是 log⁡n\log nlogn 数量级的。

canonical set: subset contained in the subtree of a node of the BST

规范集：BST 中特定节点的子树中包含的元素集。它包括节点子树中的所有元素。

更简单一点的解释就是，规范集是一颗完整的子树。

vvv highest node that splits the interval

cover the part of the interval to left and right of vvv with O(log⁡n)O(\log n)O(logn) canonical sets

left www child of vvv:

  1. outside interval: move right
  2. inside interval: move left, add right can. set to cover.

cover the part of the interval to left and right of vvv with O(log⁡n)O(\log n)O(logn) canonical sets

total size of the canonical sets: O(nlog⁡n)O(n\log n)O(nlogn)

Range Trees

input: set PPP of nnn points in 2D

goal: process PPP in a data structure

query: axis-alligned rectangle rrr

building the DS:

  1. a BST TTT on PPP ordered by x-coordinates
  2. a BST T(S)T(S)T(S) for every canonical set SSS of TTT ordered by y-coordinates

树套树：二叉搜索树套二叉搜索树？

T(S)T(S)T(S) is called a secondary DS.

canonical sets of T(S)T(S)T(S) are called secondary canonical sets.

answering query r=[x1,x2]×[y1,y2]r=[x_1,x_2]\times[y_1,y_2]r=[x1​,x2​]×[y1​,y2​]:

  1. cover [x1,x2][x_1,x_2][x1​,x2​] with l=O(log⁡n)l=O(\log n)l=O(logn) canonical sets S1,⋯ ,SlS_1,\cdots,S_lS1​,⋯,Sl​ from TTT
  2. for each SiS_iSi​, cover [y1,y2][y_1,y_2][y1​,y2​] with O(log⁡n)O(\log n)O(logn) secondary canonical sets.

query time: O(log⁡2n)O(\log^2n)O(log2n) (n is number of secondary canonical sets)
