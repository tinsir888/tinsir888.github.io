Interactive Turing Machine

📖Definition 44 (interactive Turing machine). An interactive TM MMM is a probabilistic TM equipped also with a write-only query tape and a read-only answer tape, and has two new special states, a query state and a continue state. The symbols allowed on the query and answer tape are given by a communication alphabet Λ\LambdaΛ.

When MMM enters the query state, the contents mmm of the query tape as a message mmm sent by MMM. We say that we send a reply rrr to mmm by performing the following steps: The query tape is reinitialized to a blank tape, the string rrr is placed on the answer tape, and computation of MMM is resumed in the continue state.

Given two interactive TM, consider running both TM on the same input xxx and relaying messages back and forth. In this way we define interactive proof systems having one TM be the verifier VVV, assumed to be Polynomial-time bounded, and the other TM to be the prover PPP, just assume that PPP will always produce a reply to any message received.

P:Λ∗→Λ∗P:\Lambda^*\to\Lambda^*P:Λ∗→Λ∗

Given also a polynomial time interactive TM VVV, denote by (V,P)(x)(V,P)(x)(V,P)(x) the computation of VVV on input xxx where the replies to messages of VVV are determined as follows.

Suppose that VVV enters query states and m1,⋯ ,mim_1,\cdots,m_im1​,⋯,mi​ is the list of all messages sent by VVV. The reply given to the message mim_imi​ is P(⟨x,m1,⋯ ,mi⟩)P(\langle x,m_1,\cdots,m_i\rangle)P(⟨x,m1​,⋯,mi​⟩).

The Sumcheck Protocol

How to utilize the power of interactive proofs? Converting Boolean formulas to arithmetic formulas and exploit that polynomials can be evaluated on inputs outside the Boolean cube.

Arithmetization: The process of converting a Boolean formula into an arithmetic formula.

By arithmetization, we can prove that P#P⊆IPP^{\#P}\subseteq IPP#P⊆IP, where #P\#P#P is a class capturing the complexity of counting the number of satisfying assignments of a Boolean formula.

A weaker result: coNP⊆IPcoNP\subseteq IPcoNP⊆IP.

The method can be generalized to show PSPACE⊆IPPSPACE\subseteq IPPSPACE⊆IP.
