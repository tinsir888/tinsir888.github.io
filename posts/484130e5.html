GNN - graph neural networks

Intuition: how to propagate information over the neighbors in a non-linear manner.

Graph Embeddings as Matrix Factorization

Linear embeddings correspond to SVD of the similarity matrices chosen in the objective.

For adjacency-based similarity, the best linear ddd-dimensional embedding corresponds to the projection of the dataset into the ddd directions associated to the top-ddd singular values:

if SVD(A)=UΣV⊤then the embedding matrix Z=UdΣd\text{if }SVD(\mathbf A)=\mathbf U\Sigma\mathbf V^\top\\ \text{then the embedding matrix }\mathbf Z=\mathbf U_d\Sigma_dif SVD(A)=UΣV⊤then the embedding matrix Z=Ud​Σd​

Question: Does there exist a matrix that, once factorized, would be approximately equivalent to run gradient descent over the non-linear objective?

NetMF

DeepWalk, one of the most popular random-walk embedding methods.

A random-walk approach samples random walks using a certain strategy RRR.

A random-walk embedding method optimizes the likelihood that a node jjj is reachable from a random-walk starting from node iii.

DeepWalk’s strategy is to generate random-walks of length LLL and then generate pairs of co-occuring nodes in a window of size TTT. The co-occurences are stored in a multiset D\mathcal DD.

Pseudo-code:

  • for each n=1,2,⋯ ,Nn=1,2,\cdots,Nn=1,2,⋯,N do
    • pick w1nw_1^nw1n​ according to probability distribution p(w1)p(w_1)p(w1​)
    • generate vertex sequence (w1n,⋯ ,wLn)(w_1^n,\cdots,w_L^n)(w1n​,⋯,wLn​) of length LLL by random walk on network GGG
    • for each j=1,2,⋯ ,L−Tj=1,2,\cdots,L-Tj=1,2,⋯,L−T do
      • for each r=1,⋯ ,Tr=1,\cdots,Tr=1,⋯,T do
        • add vertex-context pair (wjn,wj+rn)(w_j^n,w_{j+r}^n)(wjn​,wj+rn​) to multi-set D\mathcal DD
        • add vertex-context pair (wj+rn,wjn)(w_{j+r}^n,w_{j}^n)(wj+rn​,wjn​) to multi-set D\mathcal DD

The first vertex of the random-walk w1nw_1^nw1n​ is sampled from a prior distribution p(w1)p(w_1)p(w1​) that in undirected, connected bipartite graphs correspond to p(i)=di/2∣E∣p(i)=d_i/2|E|p(i)=di​/2∣E∣.

This process and the relative transformation through softmax corresponds to a similarity matrix. We can represent the similarity among two nodes i,ji,ji,j as

\log\left(\frac{\text{(# of RW between i and j)}\cdot\text{(# of node/context pairs)}}{k\text{(# of occurrences of i)}\cdot\text{(# of occurrences of j)}}\right)\\ =\log\left(\frac{\#(i,j)|\mathcal D|}{k\#(i)\#(j)}\right)

where kkk is the number of negative samples.

Understand this formula:

  • First partition the set of multi-set D\mathcal DD of pairs of nodes co-occurring in walks into two set Dr→\mathcal D_{r\rarr}Dr→​ and Dr←\mathcal D_{r\larr}Dr←​

    Dr→={(w,c):(w,c)∈D,w=wjn,c=wj+rn}Dr←={(w,c):(w,c)∈D,c=wj+rn,w=wjn}\mathcal D_{r\rarr}=\{(w,c):(w,c)\in\mathcal D,w=w_j^n,c=w_{j+r}^n\}\\ \mathcal D_{r\larr}=\{(w,c):(w,c)\in\mathcal D,c=w_{j+r}^n,w=w_j^n\}Dr→​={(w,c):(w,c)∈D,w=wjn​,c=wj+rn​}Dr←​={(w,c):(w,c)∈D,c=wj+rn​,w=wjn​}

    log⁡(#(i,j)∣D∣k#(i)#(j))=log⁡(#(i,j)÷∣D∣k(#(i)÷∣D∣)(#(j)÷∣D∣))#(i,j)÷∣D∣=12T∑r=1T(#(i,j)r→∣D∣r→+#(i,j)r←∣D∣r←)\log\left(\frac{\#(i,j)|\mathcal D|}{k\#(i)\#(j)}\right)=\log\left(\frac{\#(i,j)\div|\mathcal D|}{k(\#(i)\div|\mathcal D|)(\#(j)\div|\mathcal D|)}\right)\\ \#(i,j)\div|\mathcal D|=\frac{1}{2T}\sum_{r=1}^T\left(\frac{\#(i,j)_{r\rarr}}{|\mathcal D|_{r\rarr}}+\frac{\#(i,j)_{r\larr}}{|\mathcal D|_{r\larr}}\right)log(k#(i)#(j)#(i,j)∣D∣​)=log(k(#(i)÷∣D∣)(#(j)÷∣D∣)#(i,j)÷∣D∣​)#(i,j)÷∣D∣=2T1​r=1∑T​(∣D∣r→​#(i,j)r→​​+∣D∣r←​#(i,j)r←​​)

    where the latter observation comes from ∣Dr→∣/∣D∣=∣Dr←∣/∣D∣=12T|\mathcal D_{r\rarr}|/|\mathcal D|=|\mathcal D_{r\larr}|/|\mathcal D|=\frac{1}{2T}∣Dr→​∣/∣D∣=∣Dr←​∣/∣D∣=2T1​.

  • The observe what happens as the length of random walk L→∞L\rarr\infty%L→∞. We first define the random walk matrix P=Δ−1A\mathbf P=\mathbf\Delta^{-1}\mathbf AP=Δ−1A. The volume of a graph GGG is the sum of the entries of adjacency matrix vol(G)=∑i∑jai,j\text{vol}(G)=\sum_i\sum_ja_{i,j}vol(G)=∑i​∑j​ai,j​. There are 3 important Lemmas:

    \frac{\#(i,j)_{r\rarr}}{|\mathcal D|_{r\rarr}}\stackrel{p}\rarr\frac{d_i}{\text{vol(G)}}(\mathbf P^r)_{i,j}\and \frac{\#(i,j)_{r\larr}}{|\mathcal D|_{r\larr}}\stackrel{p}\rarr\frac{d_i}{\text{vol(G)}}(\mathbf P^r)_{j,i}\\ \frac{\#(i,j)}{|\mathcal D|}\stackrel{p}\rarr\frac{1}{2T}\sum_{r=1}^T\left(\frac{d_i}{\text{vol(G)}}(\mathbf P^r)_{i,j}+\frac{d_i}{\text{vol(G)}}(\mathbf P^r)_{j,i}\right)\\ \frac{\#(i,j)|D|}{\#(w),\#(c)}\stackrel{p}\rarr\frac{\text{vol}(G)}{2T}\left(\frac{1}{d_j}\sum_{r=1}^T(\mathbf P^r)_{i,j}+\frac{1}{d_i}\sum_{r=1}^T(\mathbf P^r)_{j,i}\right)

  • Putting all together we obtain DeepWalk with infinite length random walks converges to

    log⁡(vol(G)k(1T∑r=1T(Δ−1A)r)Δ−1)\log\left(\frac{\text{vol}(G)}{k}(\frac{1}{T}\sum_{r=1}^T(\mathbf\Delta^{-1}\mathbf A)^r)\mathbf\Delta^{-1}\right)log(kvol(G)​(T1​r=1∑T​(Δ−1A)r)Δ−1)

By factorizing such a similarity with SVD we can obtain the desired result that approximates the embeddings of a random-walk approach.

Graph Neural Network

abbr GNN

An architecture that allows nodes to propagate information to the neighbors in a non-linear manner. We have an attributed graph G=(V,E,X)G=(V,E,\mathbf X)G=(V,E,X) with matrix of node features X∈Rm×n\mathbf X\in\mathbb R^{m\times n}X∈Rm×n. (such a matrix encodes nodes attributes such as labels, profiles or characteristics)

The main ingredient of a GNN is the aggregation function that takes in input the features from the neighbors of a node iii and finds an embedding hi\mathbf h_ihi​.

Neighborhood Aggregation

Main idea: Generate node embeddings based on local neighborhoods of each node, using neural networks to represent the complex relationships of a graph.

Intuitively, think of the neighbors as a computational graph of a Neural Network.

The embeddings in the layer lll is an aggregation of the embeddings in layer l−1l-1l−1.

How to aggregate information across the layers? A simple approach: average neighbor information and apply a neural network.

The embeddings of the initial layer 000 are the attributes zi0=zi\mathbf z_i^0=\mathbf z_izi0​=zi​. The lll-th layer embedding of iii is

zil=σ(Wl∑j∈N(i)zjl−1∣N(i)∣+Blzil−1),∀t>0\mathbf z_i^l=\sigma(\mathbf W_l\sum_{j\in N(i)}\frac{\mathbf z_j^{l-1}}{|N(i)|}+\mathbf B_l\mathbf z_i^{l-1}),\forall t\gt0zil​=σ(Wl​j∈N(i)∑​∣N(i)∣zjl−1​​+Bl​zil−1​),∀t>0

σ\sigmaσ is non-linear activation function. The sum is the average of each neighbor’s embeddings compute in layer l−1l-1l−1.

Loss function. W,B\mathbf {W,B}W,B are the trainable weight matrices. After kkk-layers of neighborhood aggregation, we get output embeddings for each node. Loss function needs non-linear and differentiable.

We optimize the parameters of model through stochastic gradient descent SGD.

Rewrite model in matrix notation

Zl+1=σ(ZlWl+A~ZlW1l)\mathbf Z^{l+1}=\sigma(\mathbf Z^l\mathbf W^l+\tilde{\mathbf A}\mathbf Z^l\mathbf W^l_1)Zl+1=σ(ZlWl+A~ZlW1l​)

where A~=Δ−1/2AΔ−1/2\tilde{\mathbf A}=\mathbf\Delta^{-1/2}\mathbf {A\Delta}^{-1/2}A~=Δ−1/2AΔ−1/2. To optimize the parameters, a cross-entropy loss function would work in a supervised setting w.r.t. node’s labels.

An important properties of models: ability to share the parameters W,B\mathbf {W,B}W,B and allow for generalization to unseen nodes, edges or graphs.

Depend on loss function and embedding matrix, the GNN can learn node embeddings, graph embeddings or clusters.

Graph Convolutional Networks

abbr. GCN

A different neighbor aggragation

zil=σ(Wl∑j∈N(i)∪izjl−1∣N(i)∣∣N(j)∣).\mathbf z_i^l=\sigma\left(\mathbf W_l\sum_{j\in N(i)\cup i}\frac{\mathbf z_j^{l-1}}{\sqrt{|N(i)||N(j)|}}\right).zil​=σ⎝⎛​Wl​j∈N(i)∪i∑​∣N(i)∣∣N(j)∣​zjl−1​​⎠⎞​.

The main idea is node with a large number of connections should be less important. There is no need for the bias term B\mathbf BB and the per-neighbor normalization.

Normalization empirically attains better results and better parameter sharing. We can efficiently compute the embeddings using sparse batch operations

Zl+1=σ(Δ−1/2A~Δ−1/2ZlWl)\mathbf Z^{l+1}=\sigma(\mathbf\Delta^{-1/2}\tilde{\mathbf A}\mathbf\Delta^{-1/2}\mathbf Z^l\mathbf W_l)Zl+1=σ(Δ−1/2A~Δ−1/2ZlWl​)

where A~=A+I\tilde{\mathbf A}=\mathbf A+\mathbf IA~=A+I and δ\deltaδ is the diagonal degree matrix. The time complexity to propagate the information is O(∣E∣)\mathcal O(|E|)O(∣E∣).

GraphSAGE

Instead of fixed aggregation, GraphSAGE use any differentiable function that maps set of vectors to a single vector. So, if we use a generalized aggregation, we get

zil=σ([Wl⋅AGG({zjl−1,∀j∈N(i)}),Blzil−1])\mathbf z_i^l=\sigma([\mathbf W_l\cdot AGG(\{\mathbf z_j^{l-1},\forall j\in N(i)\}),\mathbf B_l\mathbf z_i^{l-1}])zil​=σ([Wl​⋅AGG({zjl−1​,∀j∈N(i)}),Bl​zil−1​])

Concentrate self embedding of neighbor embedding. Also we use a generalized aggregation.

    where we take element-wise mean/max
