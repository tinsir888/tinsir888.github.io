Fair AI/ML 3 PROP Fair Clustering Revisited

Created2024-10-10|Updated2024-10-10|AUDatalogi算法博弈论AI/ML 公平性
|Word Count:3.7k|Reading Time:22mins|Post Views:

Read the paper Proportionally Fair Clustering Revisited by Micha et Shah.

kkk cluster centers must be placed given nnn points in a metric space.

The cost to each point is its distance to the nearest cluster center.

This paper: Focus on the case where cluster centers can be placed anywhere in metric space.

E.g. L2L^2L2 distance over Rt\mathbb R^tRt, the approximation ratio of greedy capture improves to 222.

For L1,L∞L^1,L^\inftyL1,L∞ metric, the approximation ratio remains 1+21+\sqrt 21+2​.

The paper gives universal lower bounds which apply to all algorithms.

Introduction

Points N\mathcal NN, possible cluster centers M\mathcal MM.

Given k∈Nk\in\mathbb Nk∈N, task is to select a set X⊆MX\subseteq\mathcal MX⊆M consisting of ∣X∣=k|X|=k∣X∣=k cluster centers.

Group fairness.

Proportional fairness.

Contribution

Focus on the case where the metric consists of usual distance functions such as L1,L2,L∞L^1,L^2,L^\inftyL1,L2,L∞ over Rt\mathbb R^tRt.

⚠️Cluster centers can be placed anywhere in the infinite metric space. M=Rt\mathcal M=\mathbb R^tM=Rt❗

请注意此处，候选聚类中心和上一篇论文不一样，所以得出的近似下界不一样。

Greedy Capture by Chen et al. where d∈{L1,L2,L∞}d\in\{L^1,L^2,L^\infty\}d∈{L1,L2,L∞} and M=Rt\mathcal M=\mathbb R^tM=Rt.

For d=L2,M=Rtd=L^2,\mathcal M=\mathbb R^td=L2,M=Rt, GC provides 222-PROP. The approximation is obtained by Apollonius radius.

For L1,L∞L^1,L^\inftyL1,L∞, the approximation ratio can not be better than 1+21+\sqrt21+2​.

Universal lower bounds: for d=L2d=L^2d=L2 and M=Rt\mathcal M=\mathbb R^tM=Rt, no algorithm achieves better than 23≈1.155\frac{2}{\sqrt 3}\approx1.1553​2​≈1.155-PROP, whereas for d∈{L1,L∞}d\in\{L^1,L^\infty\}d∈{L1,L∞} and M=Rt\mathcal M=\mathbb R^tM=Rt, the lower bound is 1.41.41.4.

Also, this paper consider the case where M\mathcal MM is the set of nodes of an unweighted graph, and ddd measures the shortest distance between two nodes on the graph.

  • If the graph is a tree, then PROP must exist. And there are an efficient algorithm.
  • If the graph is arbitrary, k≥n/2k\ge n/2k≥n/2. PROP must exist and can be computed efficiently.
  • Open problem: Whether an exact PROP exists for all graphs.

For d=L2d=L^2d=L2 and M=Rt\mathcal M=\mathbb R^tM=Rt for t≥2t\ge2t≥2, checking whether a clustering is PROP is NP-hard.

When ttt is large,even greedy capture is NP-hard. When ttt is constant, the problem becomes efficiently solvable.

When ttt is large, using a PTASPTASPTAS as sub-routine of greedy capture, we can efficiently compute 2(1+ϵ)2(1+\epsilon)2(1+ϵ)-PROP fair clustering for any fixed ϵ>0\epsilon\gt0ϵ>0

Problem of generalization: Would a clustering that is PROP w.r.t. samples drawn from N\mathcal NN remain approximately PROP w.r.t. entire set N\mathcal NN? Chen et al. proved that the answer is yes when ∣M∣|\mathcal M|∣M∣ is finite. Using the VC dimension, the answer is also yes for M=Rt\mathcal M=\mathbb R^tM=Rt.

What about Non-tree

Consider the universal lower bound for (R2,L1)(\mathbb R^2,L^1)(R2,L1) metric from Theorem 8.

If the graph is dense enough, we can derive same lower bound of 1.41.41.4.

❓Whether better lower bounds exist is an open problem.

❓For special case where N=V\mathcal N=VN=V, we do not know whether PROP always exists

If GGG is connected and we want to place a large number of clusters k≥n/2k\ge n/2k≥n/2, then PROP always exists. This is because a dominating set of any size k≥n/2k\ge n/2k≥n/2 is guaranteed to exist in a graph with nnn nodes and can be computed efficiently.

Definition of dominating set: A set of nodes is called a dominating set if every node in the graph is either in this set or adjacent to a node in this set.

Thus the problem of finding a PROP clustering in general graphs with N=M=V\mathcal N=\mathcal M=VN=M=V is trivial when k≥n/2k\ge n/2k≥n/2, but remains open when k<n/2k\lt n/2k<n/2.

Computational Aspects

Two problems:

  1. Checking whether a PROP clustering exits.
  2. Implementing the Greedy Capture when M=Rt\mathcal M=\mathbb R^tM=Rt.

🎯Theorem 10, Given a finite N\mathcal NN, finite M\mathcal MM, k∈Nk\in\mathbb Nk∈N, and d=L2d=L^2d=L2, checking whether a PROP clustering exists is NP-hard.

Polynomial-time reduction from the planar monotone rectilinear 3-SAT problem.

Planar monotone rectilinear 3-SAT problem: Given a 3-SAT formula, each clause cjc_jcj​ consists of only positive or only negative literals, the graph connecting clauses to literals they contain is planar, and this graph has a planar embedding in which each variable viv_ivi​ is represented by a rectangle on the xxx-axis and each positive/negative clause is represented by a rectangle above/below xxx-axis with 3 vertical lines or legs to its 3 variables.
