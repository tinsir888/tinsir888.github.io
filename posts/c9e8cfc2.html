Summary of Complexity Classes

  1. L (Logarithmic Space)
    • Definition: LLL consists of all decision problems that can be solved by a deterministic Turing machine using O(log⁡n)O(\log n)O(logn) space.
    • Example: Determining if a path exists between two nodes in an undirected graph using depth-first search with a space-efficient representation.
  2. NL (Non-deterministic Logarithmic Space)
    • Definition: NLNLNL consists of all decision problems that can be solved by a non-deterministic Turing machine using O(log⁡n)O(\log n)O(logn) space.
    • Example: Determining if a path exists between two nodes in a directed graph (also known as the reachability problem).
  3. P (Polynomial Time)
    • Definition: PPP consists of all decision problems that can be solved by a deterministic Turing machine in polynomial time, O(nk)O(n^k)O(nk) for some constant kkk.
    • Example: Sorting an array using algorithms like Merge Sort or Quick Sort.
  4. NP (Non-deterministic Polynomial Time)
    • Definition: NPNPNP consists of all decision problems for which a given solution can be verified in polynomial time by a deterministic Turing machine. Equivalently, problems solvable by a non-deterministic Turing machine in polynomial time.
    • Example: The Boolean satisfiability problem (SAT).
  5. PSPACE (Polynomial Space)
    • Definition: PSPACEPSPACEPSPACE consists of all decision problems that can be solved by a deterministic Turing machine using polynomial space.
    • Example: Generalized games like chess, checkers, and go with a polynomial bound on the number of moves.
  6. EXP (Exponential Time)
    • Definition: EXPEXPEXP consists of all decision problems that can be solved by a deterministic Turing machine in exponential time, O(2p(n))O(2^{p(n)})O(2p(n)) for some polynomial p(n)p(n)p(n).
    • Example: Solving the traveling salesman problem using a brute-force search over all possible paths.
  7. NEXP (Non-deterministic Exponential Time)
    • Definition: NEXPNEXPNEXP consists of all decision problems solvable by a non-deterministic Turing machine in exponential time, O(2p(n))O(2^{p(n)})O(2p(n)) for some polynomial p(n)p(n)p(n).
    • Example: Certain advanced combinatorial problems that require guessing an exponentially long solution and verifying it in exponential time.

We don’t know whether these inclusions are strict. But we usually assume that all the inclusions are strict.
