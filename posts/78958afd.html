Potential functions

Definition: Function Φ:S→R\Phi:\mathbf S\rightarrow\mathbb RΦ:S→R is a potential function if for every two states S1S_1S1​ and S2S_2S2​ that differ in the strategy of a single player iii, the differences Φ(S1)−Φ(S2)\Phi(S_1)-\Phi(S_2)Φ(S1​)−Φ(S2​) and costi(S1)−costi(S2)\text{cost}_i(S_1)-\text{cost}_i(S_2)costi​(S1​)−costi​(S2​) have the same sign.

Potential games: those admitting a potential function

Theorem: Every potential game have at least one pure Nash equilibrium

Proof

Consider the Nash dynamics graph which has a node for each state of the game and a directed edge from state SSS to state S′S'S′ if they differ in the strategy of a single player iii, who improves cost when deviating from the strategy she uses in state SSS to the strategy she uses in state S′S'S′.

By definition of the potential function Φ\PhiΦ, the potential value Φ(S)\Phi(S)Φ(S) is strictly higher than Φ(S′)\Phi(S')Φ(S′). Then, the Nash dynamics graph can not have directed cycles. This implies that there will be a node which is a sink i.e., it has only incoming edges. Let SSS be such the state corresponding to this node. Then, no player has an incentive to deviate from SSS. Thus, SSS is a pure Nash equilibrium.
