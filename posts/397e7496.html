Core

If no group of agents S⊆NS\subseteq NS⊆N can choose another outcome that

  1. They are entitled to choose based on their proportion of the whole population ∣S∣/∣N∣|S|/|N|∣S∣/∣N∣
  2. Every member of group SSS is happier

📖Definition 1 (α\alphaα-Core). For α≥1\alpha\ge1α≥1, a kkk-clustering C=(C1,⋯ ,Ck)C=(C_1,\cdots,C_k)C=(C1​,⋯,Ck​) is said to be in the α\alphaα-core if there is no group of agents S⊆NS\subseteq NS⊆N with ∣S∣≥n/k|S|\ge n/k∣S∣≥n/k s.t. α⋅ℓi(S)<ℓi(C(i))\alpha\cdot\ell_i(S)\lt\ell_i(C(i))α⋅ℓi​(S)<ℓi​(C(i)) for all i∈Si\in Si∈S. 111-core is the core.

If a group SSS exists that becomes a violation of the α\alphaα-core guarantee, we say that it deviates and refer to it as the deviating coalition.

🎯Theorem 1. For arbitrary losses, there exists an instance in which no α\alphaα-core clustering for any finite α\alphaα.

🎯Theorem 2. For the average loss, there exists an instance in which no α\alphaα-core clustering exists an instance in which no α\alphaα-core clustering exists for α<1+32≈1.366\alpha\lt\frac{1+\sqrt3}{2}\approx1.366α<21+3​​≈1.366.

❓Open problem 1: For the maximum loss, does there always exist a clustering in the core?

❓Open problem 2: For the average loss, does there always exist a clustering in the α\alphaα-core for some constant α\alphaα?

GreedyCohesiveClustering Algorithm

It uses a subroutine A\mathcal AA, in which given a subset of agents N′⊆NN'\subseteq NN′⊆N, metric ddd, and threshold τ\tauτ, finds a cohesive cluster SSS.

The threshold τ\tauτ indicate the smallest size at which a group of agents deserve to form a cluster.

A\mathcal AA is SmallestAgentBall. It finds the smallest ball centered at agent that captures at least τ\tauτ agents, and returns a set of τ\tauτ agents from this ball.

  • Natural choice: τ=⌈n/k⌉\tau=\lceil n/k\rceilτ=⌈n/k⌉

So GreedyCohesiveClustering with SmallestAgentBall iteratively finds the smallest agent-centered ball containing ⌈n/k⌉\lceil n/k\rceil⌈n/k⌉ agents and removes ⌈n/k⌉\lceil n/k\rceil⌈n/k⌉ in that ball, until fewer than ⌈n/k⌉\lceil n/k\rceil⌈n/k⌉ agents remain, at which point all remaining agents are put into one cluster and any remaining clusters are left empty.

Actually the GreedyCohesiveClustering is an adaptation of Greedy Capture by Chen et al. There are two key difference:

  1. Growing balls centered at the agents instead of all feasible candidate centers.
  2. Stop a ball as soon as it captures ⌈n/k⌉\lceil n/k\rceil⌈n/k⌉ agents. While in GC, it continues.

🎯Theorem 3. For the average (resp. maximum) loss, the Greedy Capture algorithm is guaranteed to return a clustering in the (2⌈n/k⌉−3)(2\lceil n/k\rceil -3)(2⌈n/k⌉−3)-core (resp. 222-core) in O(kn)O(kn)O(kn) time complexity, and the bounds are tight.

Fully Justified Representation

Peters et al. introduced FJR as a relaxation of the core in the context of approval-based committee selection.

📖 Definition 2 (α\alphaα-FJR). For α≥1\alpha\ge1α≥1, a kkk-clustering CCC satisfies α\alphaα-FJR if there is no group of agents S⊆NS\subseteq NS⊆N with ∣S∣≥n/k|S|\ge n/k∣S∣≥n/k s.t. α⋅ℓi(S)<min⁡j∈Sℓj(C(j))\alpha\cdot\ell_i(S)\lt\min_{j\in S}\ell_j(C(j))α⋅ℓi​(S)<minj∈S​ℓj​(C(j)) for each i∈Si\in Si∈S, i.e., if α⋅max⁡i∈Sℓi(S)<min⁡j∈Sℓj(C(j))\alpha\cdot\max_{i\in S}\ell_i(S)\lt\min_{j\in S}\ell_j(C(j))α⋅maxi∈S​ℓi​(S)<minj∈S​ℓj​(C(j)). We refer to 111-FJR as FJR.

🤔Proposition 1. For α≥1\alpha\ge1α≥1, α\alphaα-core implies α\alphaα-FJR for arbitrary loss functions.

Trivially, ℓi(C(i))≥min⁡j∈Sℓj(C(j))\ell_i(C(i))\ge\min_{j\in S}\ell_j(C(j))ℓi​(C(i))≥minj∈S​ℓj​(C(j)).

Arbitrary Loss Functions

FJR clustering is guaranteed to exist even for arbitrary losses.

📖Definition 3 (most cohesive cluster). Given a set of agents NNN and a threshold τ\tauτ, the MostChesiveCluster problem asks to find a cluster S⊆NS\subseteq NS⊆N of size at least τ\tauτ s.t. the maximum loss of any i∈Si\in Si∈S for SSS is minimized, i.e., find arg⁡min⁡S⊆N′:∣S∣≥τmax⁡i∈Sℓi(S)\arg\min_{S\subseteq N':|S|\ge\tau}\max_{i\in S}\ell_i(S)argminS⊆N′:∣S∣≥τ​maxi∈S​ℓi​(S).

Average and Maximum Loss Functions

Let A∗\mathcal A^*A∗ be an exact ALG for the MostCohesiveCluster problem for the average loss.

It may run not in polynomial time. It can be used to detect whether a given undirected graph admits a clique of at least a given size (NP-Complete).

🤔Lemma 1. For the average (resp. maximum) loss, SmallestAgentBall is a 444-approximation (resp. 222-approximation) algorithm for the MostCohesiveCluster problem, and this is tight.

🤔Corollary 1. The Greedy Capture algorithm is guaranteed to return a clustering that is 444-FJR (resp. 222-FJR) for the average (resp. maximum) loss.

❓Open problem 3: For the average or maximum loss, what is the smallest α\alphaα for which an α\alphaα-FJR clustering can be computed in polynomial time, assuming P≠NPP\neq NPP=NP?

❓Open problem 4: What is the smallest α\alphaα s.t. there always exists a clustering that is simultaneously α\alphaα-FJR for both the average loss and the maximum loss?

Auditing FJR

Ideas used to find an approx FJR can also be used to audit the FJR approx of any given clustering.

📖Definition 4 (λ\lambdaλ-approx FJR auditing). A\mathcal AA is a λ\lambdaλ-approx FJR auditing algorithm, if given any clustering CCC, it returns θ\thetaθ s.t. the exact FJR approx of CCC is in [θ,λ⋅θ][\theta,\lambda\cdot\theta][θ,λ⋅θ].

🎯Theorem 5. For λ≥1\lambda\ge1λ≥1, if A\mathcal AA is a λ\lambdaλ-approx algorithm to the MostCohesiveCluster problem, then AuditFJR(A)(\mathcal A)(A) is a λ\lambdaλ-approx FJR auditing algorithm. Given lemma 1, it follows that for the average (resp. maximum) loss, AuditFJR(SmallestAgentBall) is an efficient 444-approx (resp. 222-approx) FJR auditing algorithm.

Discussion

Introduction to Proportional Fairness in Non-Centroid Clustering: This paper initiates the study of proportional fairness in non-centroid clustering and presents several open questions for further exploration.

Key Open Questions:

  • Achieving a better approximation than O(n/k)O(n/k)O(n/k) for the core concerning average loss.
  • Determining if the core is always non-empty for maximum loss.

Core Non-Emptiness Results:

  • Proven that the core is always non-empty for maximum loss in 1-dimensional metric spaces (Appendix C.1).
  • For average loss, however, the core remains empty even on the line (Appendix C.2).

Differences Between Clustering Settings:

  • Highlights significant distinctions between centroid and non-centroid clustering.

Future Directions:

  • Investigate proportional fairness guarantees when an agent’s loss depends on both their cluster center and other agents in their cluster.
  • Explore the possibility of intrinsically choosing the number of clusters kkk, which currently requires being fixed in advance for proportional fairness.

Compatibility with Classical Algorithms:

  • While traditional algorithms like kkk-means and kkk-centers may be incompatible with the core and FJR (Appendix E), studying conditions under which they might align with fairness criteria or can be efficiently computed is an interesting avenue.
