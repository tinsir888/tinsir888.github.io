Boolean satisfiability is prototypical NPNPNP-complete problem.

There is no algorithm deciding SATSATSAT in worst case in polynomial time unless P=NPP=NPP=NP.

We always assume that P≠NPP\neq NPP=NP.

For SATSATSAT problem, SOTA algorithm uses time 2(1−1O(log⁡(m/n)))nmO(1)2^{(1-\frac{1}{O(\log(m/n))})n}m^{O(1)}2(1−O(log(m/n))1​)nmO(1).

For special case of kSATkSATkSAT, there are faster algorithms, but all take time 2(1−ck)n2^{(1-\frac{c}{k})n}2(1−kc​)n for some constant 0<c<k0\lt c\lt k0<c<k.

Schöning’s kSAT Algorithm

for all x∈{0,1}nx\in\{0,1\}^nx∈{0,1}n.

Reduce to Hitting Set Problem

Recall HittingSetHittingSetHittingSet problem. Consider a universe UUU of elements and a collection S\mathcal SS of subsets of U\mathcal UU. We say that T⊆UT\subseteq\mathcal UT⊆U is a hitting set for S\mathcal SS if S∩T≠∅S\cap T\neq\emptyS∩T=∅ for all S∈SS\in\mathcal SS∈S. We denote by σ(S)\sigma(\mathcal S)σ(S) the collection of all hitting sets of S\mathcal SS.

kHittingSetkHittingSetkHittingSet problem:

  • Instance: Collection S\mathcal SS of subsets of a universe U=2U\mathcal U=2^UU=2U.
  • Output: Hitting set TTT for S\mathcal SS minimizing ∣T∣|T|∣T∣.

We denote by kHittingSetkHittingSetkHittingSet the restriction to collections of subsets of size at most kkk.

We can transform kSATkSATkSAT into kHittingSetkHittingSetkHittingSet problem:

  • C\mathcal CC is the subsets of U\mathcal UU corresponding to the clauses of Φ\PhiΦ.
  • An assignment to the variables of Φ\PhiΦ can be viewed as a subset of U\mathcal UU.
  • A\mathcal AA is the collection of the nnn sets {xi,xi‾}\{x_i,\overline{x_i}\}{xi​,xi​​} for i=1,⋯ ,ni=1,\cdots,ni=1,⋯,n. Note that assignments to the variables of Φ\PhiΦ are in one-to-one correspondence to hitting sets of A\mathcal AA.
  • Defining SΦ=C∪A\mathcal S_\Phi=\mathcal C\cup\mathcal ASΦ​=C∪A

Observation: There is a one-to-one correspondence between the set of satisfying assignments of Φ\PhiΦ and hitting sets of SΦ\mathcal S_\PhiSΦ​ of size nnn.

Any collection of subsets T\mathcal TT over U\mathcal UU can be viewed as a CNF formula by letting each subset define a clause.

📖Definition 57. Let T\mathcal TT and S\mathcal SS be collections of subsets. We say that T\mathcal TT is a restriction of S\mathcal SS if for every S∈SS\in\mathcal SS∈S there exists T∈TT\in\mathcal TT∈T s.t. T⊆ST\subseteq ST⊆S.

Clearly, if T\mathcal TT is a restriction of S\mathcal SS, then σ(T)⊆σ(S)\sigma(\mathcal T)\subseteq\sigma(\mathcal S)σ(T)⊆σ(S). Any satisfying assignment of CNF formula given by T\mathcal TT is also a satisfying assignment of the CNF formula given by S\mathcal SS.

🤔Lemma 23 (Hitting set correspondence). For all ϵ>0\epsilon\gt0ϵ>0 and integers kkk, there exists a constant C=C(ϵ,k)C=C(\epsilon,k)C=C(ϵ,k) and an algorithm that given as input collection S\mathcal SS of subsets of size at most kkk of a universe U\mathcal UU of size nnn runs in time tnO(1)tn^{O(1)}tnO(1), where t≤2ϵnt\le 2^{\epsilon n}t≤2ϵn, and give as output a list of restrictions T1,⋯ ,Tt\mathcal T_1,\cdots,\mathcal T_tT1​,⋯,Tt​ each with at most CnCnCn subsets of U\mathcal UU which are all of size at most kkk s.t.

σ(S)=⋃i=1tσ(Ti).\sigma(\mathcal S)=\bigcup_{i=1}^t\sigma(\mathcal T_i).σ(S)=i=1⋃t​σ(Ti​).

We call a subset of U\mathcal UU of size jjj: jjj-set.

The algorithm is a branching algorithm where each leaf of the recursion tree will give a restriction of S\mathcal SS.

To branch, the algorithm looks for a large collection of subsets of the same size that has a non-empty intersection, and branches on whether or not the common intersection will be hit by the hitting set. We define such collections of sets as flowers.

📖Definition 58. Let S1,⋯ ,ScS_1,\cdots,S_cS1​,⋯,Sc​ be jjj-sets.

  • We say that these sets are a flower if H=⋂i=1cSi≠∅H=\bigcap_{i=1}^cS_i\neq\emptyH=⋂i=1c​Si​=∅.
  • We call HHH the heart of the flower and sets (S1╲H),⋯ ,(Sc╲H)(S_1\diagdown H),\cdots,(S_c\diagdown H)(S1​╲H),⋯,(Sc​╲H) the petals.
  • Note that all petals have the same size j−∣H∣j-|H|j−∣H∣, which we denote as petal-size.

定义一朵花：一组大小为 jjj 的集合，它们的交集为空。

花心：这组集合的交集。对应定义花心的大小。

花瓣：每个集合去除花心。对应定义花瓣的大小。

The algorithm will be controlled by integer parameters 2=θ0<θ1<⋯<θk−12=\theta_0\lt\theta_1\lt\cdots\lt\theta_{k-1}2=θ0​<θ1​<⋯<θk−1​. We make θi\theta_iθi​ increase rapidly as a parameter of iii.

📖Definition 59. Let S1,⋯ ,ScS_1,\cdots,S_cS1​,⋯,Sc​ be a flower of jjj-sets and petal-size iii.

  • The flower is large if c≥θic\ge\theta_ic≥θi​.
  • The flower is good if
    • it is a large flower chosen s.t. jjj is chosen minimally
    • and if more than one large flower of jjj-sets exists then chosen s.t. iii is chosen minimally.

如果一朵花含有集合的个数足够大，则称这朵花是“大”的。

如果一朵花是“大”的，并且其中包含的集合大小、花瓣大小都是最小的，则称这朵花是“好”的。

Thus any hitting set of S\mathcal SS is preserved in at least one recursive branch.

It’s easy to see that Reduce always terminates.

It remains to establish qualitative properties of Reduce: Every restriction output by Reduce is sparse, i.e., contains at most CnCnCn sets, for a constant C=C(k,ϵ)C=C(k,\epsilon)C=C(k,ϵ).

The Exponential Time Hypothesis

A stronger hypothesis than P≠NPP\neq NPP=NP: Exponential time is required to decide SATSATSAT.

Exponential time hypothesis (aka ETH): kSATkSATkSAT requires worst-case linear exponential time, for all k≥3k\ge3k≥3.

A strengthening of this hypothesis, strong exponential time hypothesis: it is not possible to save an exponential factor 2ϵn2^{\epsilon n}2ϵn for a fixed 1>ϵ>01\gt\epsilon\gt01>ϵ>0 independent of kkk

The ETH is the statement that ∀k≥3:sk>0\forall k\ge3:s_k\gt0∀k≥3:sk​>0, and SETH is the statement that s∞=lim⁡k→∞sk=1s_{\infty}=\lim_{k\to\infty}s_k=1s∞​=limk→∞​sk​=1.
